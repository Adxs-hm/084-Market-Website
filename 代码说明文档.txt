
# 零八四商城 (084 Mall) 项目文件说明

项目已构建为原生 HTML/CSS/JS 多页面架构。资源文件已按类型归类整理。

=============================================================================
1. 核心代码文件
=============================================================================

[ 页面结构 - HTML (位于 html/ 文件夹) ]
- html/index.html       : 【首页】商城入口
- html/list.html        : 【列表页】商品筛选与搜索
- html/detail.html      : 【详情页】商品详情与评价
- html/cart.html        : 【购物车】购物车结算
- html/admin.html       : 【后台】商家管理系统 (可导出 data.js)
- html/about.html       : 【关于】品牌介绍
- html/contact.html     : 【联系】联系方式
- html/help.html        : 【帮助】帮助中心

注意：根目录下的同名 .html 文件均为跳转页，实际内容位于 html/ 目录。

[ 样式与脚本 (资源目录) ]
- css/style.css         : 全局样式表
- js/script.js          : 核心交互逻辑 (含图片灯箱、购物车逻辑)
- js/data.js            : 本地数据源 (配置、商品、内容)

[ 其他资源 ]
- json/metadata.json    : 项目元数据
- tsx/index.tsx         : React 入口文件 (仅作占位)

=============================================================================
2. 媒体资源目录
=============================================================================
请将您的媒体文件直接放入以下对应文件夹，并在 admin.html 或 js/data.js 中引用：

- images/               : 存放图片 (logo, banner, 商品图)
- video/                : 存放视频 (宣传片)
- music/                : 存放音频 (背景音乐)

=============================================================================
3. 新增功能特性
=============================================================================
- **GIF 动画支持**：后台上传 GIF 图片时会自动识别并保留动画效果（不进行压缩）。
- **图片放大镜**：鼠标悬停在商品图片上时，图片会自动放大 20% (1.2倍)。
- **全屏灯箱**：点击商品图片可查看高清全屏大图。

=============================================================================
4. 注意事项
=============================================================================
- **路径引用**：由于 HTML 文件在 `html/` 子目录，引用根目录资源时使用 `../` (例如 `../images/1.png`)。JS 代码已自动处理大部分路径。
- **数据修改**：在后台修改配置后，请导出 data.js 并覆盖 js/data.js。
